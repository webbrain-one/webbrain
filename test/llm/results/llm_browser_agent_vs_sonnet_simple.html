LLM browser-agent benchmark v3 — Sonnet 4.6 as ground truth

14 models, 100 prompts each. Every other model is scored on how often its first tool pick matched anthropic/claude-sonnet-4.6's pick for the same prompt. Latency and cost are excluded because models ran on different engines and networks, and cost was only captured for a subset.

"Match vs Sonnet (all)" treats every one of the 100 cases as judged, including ones where Sonnet itself returned no tool — agreeing with Sonnet's "no tool" decision counts as a match. "Match (when Sonnet tooled)" only counts the 92 cases where Sonnet picked a tool, so the model is judged purely on tool-selection (not on knowing-when-to-decline). "Valid-name rate" flags hallucinated tool names — tools the model invoked that don't exist in the WebBrain tool set.
