不再猜测设置含义

知道该改什么，也知道什么可以不动。

从侧边栏齿轮或浏览器扩展选项打开设置。大多数人主要使用“常规”“提供商”和“权限”；其他标签页用于解决特定问题。

适合大多数人的默认值

控制项推荐值原因
执行前计划尝试计划有计划时使用；计划失败时不至于让任务卡死。
计划审查低于置信度阈值时停止不确定的计划暂停，可靠计划可继续。
审查阈值90%在监督与流畅度之间较保守的平衡。
最大代理步骤130足够处理多步骤工作；出现重复时仍应手动停止。
LLM 请求超时120 秒云模型通常更快，大型本地模型启动可能更慢。
计划任务关键操作确认开启无人值守任务仍会在关键操作前暂停。
权限闸门开启首次在网站执行特定操作前询问，并记住你的明确选择。
日常使用

常规：代理如何行动

尝试计划适合日常使用；严格计划会在无法生成有效计划时取消任务；关闭会省去一次规划调用，但也失去计划审查节点。

步骤上限统计一条消息中的工具调用步骤，不是聊天消息数。请求超时只限制模型开始响应前的等待时间，不会中断已经开始流式输出的响应。

无限步骤不等于更可靠

∞ 只会移除机械停止点，不会让模型更不容易循环。除非你正在全程监督长任务，否则保留有限上限。

高级抽屉

高级：带着明确目的再打开

设置通常保持何时修改
截图回退视觉模型可用时开启DOM 读取遗漏图表、画布或视觉内容。
网站适配器开启调试令牌使用或禁用站点专用提示。
API 变更观察器关闭研究重复 UI 操作与 API 快捷建议。
自动截图状态变化时需要更低图像成本，或每一步都要截图。
记录追踪关闭比较模型或诊断运行，并接受本地存储。
严格秘密处理个人日常使用时关闭屏幕共享或导出追踪，不希望凭据被引用。
允许访问本地网络关闭必须访问 localhost、内网或私有地址。
CAPTCHA 求解关闭你明确配置并充值 CapSolver。
日常使用

提供商：选择当前模型

一次只有一个提供商处于活动状态。正确顺序是 保存 → 测试连接 → 设为活动。

本地服务器地址、上下文窗口和提示层级见提供商与模型。

按需使用

多模态：分别配置视觉与转录

视觉可把截图交给独立的 OpenAI 兼容视觉端点，而当前提供商继续负责规划和工具调用。留空则使用当前提供商。

转录为录音提供 Whisper 兼容端点。留空时依次尝试 OpenAI、Groq、LM Studio、llama.cpp。

按需使用

记忆：决定哪些信息长期保留

  • 加密云同步面向 WebBrain Cloud 会员；支持的提供商密钥、个人资料和记忆会在上传前用同步密码加密。聊天历史与 OAuth 登录不会同步。
  • 个人资料自动填充以明文保存在当前浏览器配置中，启用后会发送给当前模型提供商。
  • 用户记忆保存稳定偏好；/memory --add 会立即保存，自动学习是可选项。
不要把记忆当作密码管理器

不要在个人资料或用户记忆中存放密码、API 密钥、令牌、恢复码等秘密。

信任决定

技能：附加指令与工具

技能可以把指令加入系统提示；含 webbrain-tools 清单的技能还可暴露 HTTP 或下载任务工具。导入本身就是信任决定，只导入你信任的来源。

日常安全

权限：WebBrain 可以在哪里做什么

保持“关键操作前询问”开启。每条永久授权都对应一个明确的“网站 + 能力”组合；允许点击并不自动允许输入、下载、上传、执行 JavaScript 或安排任务。
