第一次运行

前十分钟

  1. 打开你想处理的网页。点击 WebBrain 图标，让侧边栏与网页并排显示。
  2. 先留在“询问”模式。例如：“总结本页”“列出重要日期”“比较表格中的套餐”。询问模式可以阅读，但不会点击或输入。
  3. 需要改动网页时再切到“执行”。例如：“在搜索框输入 wireless headphones 并搜索”。在新网站进行关键操作前，WebBrain 会请求权限。
  4. 观察过程。随时使用停止按钮或 Escape 中止。遇到意外跳转、输入或重复动作，应立即停止。
  5. 有明确需求时才打开设置。内置云提供商与推荐安全默认值足够完成第一次任务。
好的提示词要写清结果

“找到退款政策，并告诉我促销商品是否适用”比“看看这个网站”更容易验证。写明目标，也写明 WebBrain 不应越过的边界。
